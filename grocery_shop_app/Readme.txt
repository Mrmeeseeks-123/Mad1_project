1. open this folder in terminal
2. create virtual environment by typing "python3 -m venv .venv" in linux terminal
3. activate virtual environment by typing "source .venv/bin/activate" in linux terminal .
4. install requirements by typing "pip install -r requirements.txt" in linux terminal.
3. run the app by typing "python3 app.py" in terminal.
4. open http://127.0.0.1:5000/ in your browser to use the application.
